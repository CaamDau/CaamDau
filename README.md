## Installation

CD is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'CD'
```
```ruby
pod 'CD/Chain'
pod 'CD/Form'
pod 'CD/Extension'
pod 'CD/Config'
```
